# Draw Rectangle 
Created on 1/31/2018

## What is it?
A piece of code, written in C language, that asks the user to input a number n. It will then generate the 3 rectangle outputs that have n lines, using for-loop.

## How to run this program on your computer locally?
#### Windows operating system 
1. Open the Command Prompt
2. Test if you have gcc installed by typing `gcc -v`. If it is not installed, please install gcc https://gcc.gnu.org/ 
3. Go to the source code directory
4. Compile the source code `gcc -o program drawWithStars.c`
5. Run your program `program`

#### Macintosh operating systems
1. Open terminal
2. Install the command-line tools: `xcode-select --install`
3. Go to the source code directory 
4. Compile the source code `gcc -Wall -o program drawWithStars.c`
5. Run your program `./program`
